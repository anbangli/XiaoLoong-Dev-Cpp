XiaoLoong Dev-C++

 

Dev-C++ is a beginner-friendly integrated development environment (IDE) for C and C++ programming language. It uses TDM-GCC as compiler and GDB as debugger, providing basic C/C++ programming environments. Besides, it also includes all standard features of similar software, including syntax highlighting, advanced code completion and insight, profiling, debugging, style formatting of your code, and editable shortcuts. In brief, it is an excellent IDE to use for programming beginners.

XiaoLoong (little Chinese dragon) Dev-C++, previously named as Banzhu Dev-C++, is an updated fork of Orwell Dev-C++. Several known bugs were fixed, components are updated, and some new features are added.

 

Latest version: v5.16 (released in November, 2021) >> Download XiaoLoong Dev-C++ v5.16

 

Features

- User-friendly, interface concise and pretty

- Support single-file development and project development.

- Automatically format source code before saving file.

- Automatically display values of parameters and variables
